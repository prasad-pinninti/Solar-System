# 🌌 Solar System Explorer

An interactive Solar System visualization built entirely with **HTML5** and **CSS3**. This project showcases the planets of our solar system with smooth animations, orbital effects, detailed planet information, and an immersive user interface.

## 🚀 Features

* Interactive planet selection menu
* Detailed information about each planet
* CSS-based animations and transitions
* 3D visual effects using CSS transforms and perspective
* Planet orbit visualization
* Responsive and visually appealing design
* Pure HTML and CSS implementation (No JavaScript)

## 🛠️ Technologies Used

* HTML5
* CSS3
* CSS Animations
* CSS Transformations
* Flexbox
* Responsive Design
